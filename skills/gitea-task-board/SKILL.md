# Skill: Gitea Task Board Manager

## Purpose
Allows agents to create, update, and manage tasks in Gitea Issues and Project Boards.

## Capabilities
- Create issues for tasks
- Assign owners
- Apply scoped labels (stage/dev, stage/test, stage/prod)
- Move issues across project board columns
- Link issues to PRs

## Requirements
- Gitea API token
- Access to repository
