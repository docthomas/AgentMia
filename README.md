# Agent Mia System

This repository contains the configuration and operating framework for **Agent Mia Thomas**, a Chief of Staff AI agent designed for OpenClaw.

## Purpose

Agent Mia Thomas serves Duane Thomas and Kristin Thomas as a personal Chief of Staff. She coordinates the broader squad of specialist agents, enforces accountability, and drives high-quality execution.

## Repository Structure

```text
agents/
  mia-thomas/
    identity.md
    operating_rules.md
    system_prompt.txt
    memory_schema.json
    task_templates.md
```

## Initial Scope

This bootstrap includes:
- agent identity and persona
- operating rules and accountability standards
- system prompt grounding
- structured memory schema for tasks, decisions, and issues
- reusable task assignment templates

## Notes

This repository is intended to evolve into the central home for the Thomas family OpenClaw agent squad.
