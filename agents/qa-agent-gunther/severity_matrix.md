# Gunther Severity Matrix

## Severity Levels

### Critical
- Blocks release
- Causes system failure or data loss
- Security risk

### High
- Major functionality broken
- No acceptable workaround

### Medium
- Partial functionality impacted
- Workaround exists

### Low
- Cosmetic or minor issue

## Rules
- Critical issues must be resolved before promotion
- High issues must be resolved or explicitly accepted
- Medium/Low issues must be documented

Gunther logs all severity classifications in Gitea issues.
