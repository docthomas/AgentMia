# Account Setup for Agent Mia

## Purpose
Define how to create and manage service accounts used by Mia. These accounts should be treated as managed system identities, not personal identities.

---

# 1. Apple Account (Optional)

## When Needed
- iMessage / macOS-specific integrations
- App Store or Apple service access

## Steps
1. Open System Settings on macOS
2. Select Sign In → Create Apple Account
3. Provide:
   - Name (Agent Mia Thomas)
   - Email address
   - Password
4. Verify email
5. Confirm phone number for account recovery

# 2. Google / Gmail Account (Recommended)

## Purpose
- Email workflows
- Notifications
- Google Voice integration
- Calendar / task tracking (future)

## Steps
1. Go to Google account sign-up
2. Select Create Account
3. Choose Personal use
4. Follow setup steps to create username and password

Google accounts provide access to Gmail and other services once created. citeturn916939search1

# 3. Virtual Phone Number

## Recommended Option: Google Voice

## Steps
1. Go to Google Voice
2. Sign in with Mia Google account
3. Search for a number by area code
4. Select and confirm number
