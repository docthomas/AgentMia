# Project Documentation Structure

## Core Documents
- docs/sprint-plan.md → Active sprint planning
- docs/timeline.md → Roadmap and timeline
- docs/prerequisites.md → System prerequisites
- docs/install-guide.md → Step-by-step install
- docs/issues.md → Known bugs/issues
- docs/suggestions.md → Feature ideas and enhancements

## Agent Docs
- agents/* → Agent definitions and rules

## Skills
- skills/* → Integration capabilities

## Principle
All work must be tracked, planned, and cross-referenced across these documents.
